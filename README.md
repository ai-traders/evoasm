# Awasm

An AIMGP (*Automatic Induction of Machine code by Genetic Programming*) engine.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'awasm'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install awasm --pre
